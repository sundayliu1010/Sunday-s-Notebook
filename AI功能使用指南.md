## 🔑 配置OpenAI API

### 步骤1: 获取OpenAI API Key

1. 访问 [OpenAI Platform](https://platform.openai.com/)
2. 注册或登录您的OpenAI账户
3. 进入 **API Keys** 页面
4. 点击 **"Create new secret key"**
5. 复制生成的API Key（格式如：`sk-xxx...`）

### 步骤2: 在应用中配置

1. 点击顶部导航栏的 **⚙️ 设置** 按钮
2. 在配置页面输入您的OpenAI API Key
3. 点击 **"保存配置"**
4. 🎉 配置完成！Hello Kitty AI已准备就绪

### 💰 API费用说明

- **新用户免费额度**: 通常有$5-18的免费额度
- **预估用量**:
  - 文本润色: ~$0.001/次
  - 文本续写: ~$0.002/次
  - 智能洞察: ~$0.003/次
  - AI聊天: ~$0.002/条消息

## 📝 如何使用AI功能

### 1. 文本润色 ✨

**用途**: 优化您笔记中的文字表达

**使用步骤**:
1. 在笔记编辑器中选中需要润色的文字
2. 点击弹出的 **"✨ 润色"** 按钮
3. 等待AI处理（约2-3秒）
4. 润色后的文字会自动替换原文

**示例**:
```
原文: 这个东西很好用
润色后: 这款产品使用体验极佳，功能实用且操作简便
```

### 2. 文本续写 ➡️

**用途**: 基于已有内容智能扩展

**使用步骤**:
1. 选中需要续写的文字段落
2. 点击弹出的 **"➡️ 续写"** 按钮
3. 等待AI生成续写内容
4. 续写内容会自动插入到选中文字之后

**示例**:
```
原文: 今天学习了Vue的基础知识
续写后: 今天学习了Vue的基础知识。Vue是一个渐进式JavaScript框架，通过响应式数据绑定和组件化开发，能够帮助我们构建现代化的Web应用程序。接下来我计划深入学习Vue Router和Vuex。
```

### 3. 智能洞察 🧠

**用途**: 深度分析笔记内容，生成洞察报告

**使用步骤**:
1. 在笔记编辑器页面点击右下角 **"🧠 智能洞察"** 按钮
2. 等待AI分析整个笔记内容
3. 查看生成的洞察报告，包括：
   - 📝 内容摘要
   - 🏷️ 关键词提取
   - 📊 内容统计
   - ❓ 思考问题

### 4. AI聊天 🌸

**用途**: 与Hello Kitty助手进行智能对话

**使用步骤**:
1. 点击右下角浮动的 **🌸** 图标
2. 在聊天框中输入您的问题
3. Hello Kitty助手会为您提供帮助

**聊天特色**:
- 🌸 Hello Kitty主题的友好语调
- 💡 提供实用的建议和回答
- 📚 支持多轮对话，记住上下文
- ✨ 带有emoji表情的温暖交流

## 🛠 故障排除

### 常见问题

**Q: AI功能无法使用？**
A: 检查是否已正确配置OpenAI API Key，点击⚙️设置查看配置状态

**Q: API调用失败？**
A: 可能的原因：
- API Key无效或已过期
- OpenAI账户余额不足
- 网络连接问题
- API请求过于频繁

**Q: 生成的结果不理想？**
A: 尝试：
- 更清晰地描述您的需求
- 提供更多上下文信息
- 重新组织输入文本

**Q: 消耗太快？**
A: 建议：
- 合理使用AI功能
- 先用少量文字测试效果
- 监控OpenAI账户余额
